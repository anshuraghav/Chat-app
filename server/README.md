# ChatApp - Server

#### Relevant Libs/Frameworks that i'm using

  - express
  - express-validator
  - mongoose
  - jsonwebtoken
  - socket.io

# Commands

Run tests
```sh
$ npm run test
```

Run dev server
```sh
$ npm install
$ npm run dev
```

### Configuration
Set your configurations in config file

```sh
:root:/app/configs/(dev/prod).json
```
